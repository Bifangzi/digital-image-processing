# 数字图像处理

#### 介绍
西安邮电大学数字图像处理实验代码

#### 软件架构
软件架构说明
本仓库脚本均在Windows11+Syslab2025a下编写并确认可正常运行
实验室环境为Linux+Syslab2024b,部分代码可能运行出错,请自行修改


#### 安装教程

1.  直接下载本仓库压缩包并解压或通过git clone到本地
2.  在Mworks.syslab中打开对应实验的文件夹
3.  待软件包加载完成后点击`运行`

#### 使用说明

1.  请不要在压缩包中尝试运行脚本
2.  不要移动各个实验文件夹下的文件位置，以免脚本中的相对路径无法索引
